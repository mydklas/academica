PROGRAM KURSUS CODING


Bali,


Grade SD Kelas 1-3


Pengenalan pada pemrograman merupakan kursus pemrograman yang ditujukan utamanya untuk anak-anak sekolah dasar dari usia 8-12 tahun menggunakan sebuah software pemrograman grafis. Para siswa mulai pemrograman dengan coding berbasis blok. Melalui puzzle-puzzle coding, proyek dan cerita, mereka diharapkan dapat memahami konsep-konsep dasar dalam matematika dan pemrograman seperti waktu, kecepatan dan persyaratan-persyaratan tertentu yang harus dipenuhi.

  • Pengenalan pemrograman untuk anak-anak membuat permainan interaktif.
  • Anak-anak akan membuat program mereka sendiri, animasi dan permainan.
  • 60 jam pembelajaran yang menyenangkan. Masing-masing level 20 jam.
  • Beberapa tantangan dan quiz coding


Apa yang dipelajari?




Bangun logika


Belajar bagaimana menggunakan logika untuk membuat program dan memecahkan permasalahan yang rumit

Konsep Matematika


Belajar tentang konsep kunci matematika dalam pemrograman dan sistem koordinat

Desain Game


Belajar aspek-aspek berbeda dari desain dan pembuatan game dan animasi


Pernyataan Bersyarat


Belajar tentang penggunaan pernyataan bersyarat untuk membuat keputusan dalam program komputer

Konsep Waktu


Belajar bagaimana menggunakan waktu untuk mengendalikan beberapa aspek dari kode yang dibuat seperti penundaan dan kecepatan

Deret dan Pengulangan ( Loops )

Belajar bagaimana menggunakan waktu untuk mengendalikan beberapa aspek dari kode yang dibuat seperti penundaan dan kecepatan.

Grade SD Kelas 4-6


Dalam pemrograman game dengan software Scratch siswa belajar konsep-konsep kunci ilmu komputer seperti deret, loop, pernyataan bersyarat dan abstraksi dengan membangun proyek mereka sendiri dari cerita yang dianimasikan dan permainan (game). Para siswa dalam kelompok umur ini menghadapi tantangan untuk belajar tata kalimat coding oleh karena itu software Scratch sangat membantu siswa sebagai jembatan dengan persiapan dan pembekalan para siswa dengan logika yang perlu dan konsep-konsep pemrograman sedemikian hingga mereka mendapatkan kemudahan untuk belajar tata kalimat coding pada tahap lebih lanjut.


Apa yang dipelajari?




Bangun logika


Belajar bagaimana menggunakan logika untuk membuat program dan memecahkan problem yang rumit

Operasi Matematika


Belajar bagaimana menggunakan operasi matematika dalam pemrograman

Desain Game


Belajar aspek-aspek berbeda dari desain dan pembuatan game dan animasi


Pernyataan Bersyarat


Belajar tentang penggunaan pernyataan bersyarat untuk membuat keputusan dalam program komputer

Variabel


Belajar tentang penyimpanan data dalam variabel dan menggunakan mereka untuk program lebih efisien

Deret dan Pengulangan ( Loops )


Belajar bagaimana menulis program cerdas dengan loop dan deret

Grade SMP Kelas 7-9


Dirancang bagi siswa yang memiliki pikiran kreatif, para siswa dalam kursus ini belajar untuk membuat kode menggunakan tata bahasa pemrograman ( syntax ) dan sebuat alat bantu software yang membantu semudah mungkin pengenalan pada bahasa pemrograman/coding berbasiskan tata kalimat/sintaks. Melalui pembuatan sketsa coding secara visual, para siswa menghasilkan seni grafis yang menakjubkan dan menggali kesempatan-kesempatan baru untuk memanfaatkan kekuatan komputasi komputer lintas disiplin ilmu. Para siswa akan dikenalkan pada konsep-konsep kunci ilmu komputer seperti fungsi dan pengulangan, yang dikenal sebagai teori desain dasar.


Apa yang dipelajari?




Sintaks Dasar


Belajar sintaks pemrograman dasar untuk menulis program cerdas, membuat game dan kerja seni digital

Seni Digital


Belajar bagaimana untuk menggunakan teknologi digital untuk membuat kerja seni yang menarik

Tipe Data


Belajar tipe-tipe data yang berbeda seperti bilangan bulat (integer), bilangan real ( floats), boolean, dan lain-lain


Operasi-operasi


Belajar bagaimana menggunakan operasi aritmatika, operasi logika dan operasi relasi dalam pemrograman

Desain Game


Belajar aspek-aspek berbeda dari desain game dan membuat game dan animasi

Peryataan Bersyarat dan Loops


Belajar bagaimana menggunakan loop dan pernyataan bersyarat untuk menulis program cerdas

Grade SMA Kelas 10-12


Dirancang untuk siapapun siswa yang memiliki keingintahuan yang tinggi akan kekuatan komputer dan teknologinya, kursus ini mengenalkan ilmu komputer dasar melalui pemrogaman sebuah rangkaian proyek yang menyenangkan dan tantangan pembuatan kode yang bermanfaat. Setelah pengenalan sintaks pemrograman dan konsep ilmu komputer , para siswa juga belajar berpikir tentang problem secara logic dan memecahkannya secara sistematis. Kursus ini juga mengenalkan pustaka coding pada para siswa, meletakkan dasar yang solid bagi lebih banyak aplikasi-aplikasi program yang sangat bermanfaat pada tingkat yang lebih tinggi


Apa yang dipelajari ?




Sintaks Tingkat Lanjut ( Advanced Syntax )


Belajar sintaks pemrograman tingkat lanjut untuk menulis kode-kode paling efisien.

Modul-modul


Menggunakan modul-modul yang berkaitan bersama dengan kode dan membuat kode lebih mudah dipahami dan digunakan. Belajar bagaimana menggunakan modul-modul secara cerdas dalam kode

Tipe Data


Belajar tipe-tipe data yang berbeda seperti bilangan bulat (integer), bilangan real ( floats), boolean, dan lain-lain


Operasi-operasi


Belajar bagaimana menggunakan operasi aritmatika, operasi logika dan operasi relasi dalam pemrograman

Peryataan Bersyarat dan Loops


Belajar bagaimana menggunakan loop dan pernyataan bersyarat untuk menulis program cerdas

Membangun Software


Belajar bagaimana membuat software sederhana yang digunakan dalam kehidupan nyata seperti chatbots.
